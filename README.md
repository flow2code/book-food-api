# book-food-api
organize your orders schedule


## installation

- `npm install`

## run

- `mongod`
- `node index.js`


## usage

### specification
`GET http://localhost:8081/api/v1/specifications`

### users
`GET http://localhost:8081/api/v1/users`

### orders
`http://localhost:8081/api/v1/collections/orders`
